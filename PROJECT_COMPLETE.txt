💡 USAGE EXAMPLES
═══════════════════════════════════════════════════════════════════════════

Web Interface:
  1. Open http://localhost:5000
  2. Drag PDF files into upload area
  3. Select processing options
  4. Click "Process Files"
  5. View results and download

Python Library:
  from pdf_processor import PDFReader
  reader = PDFReader("document.pdf")
  text = reader.extract_text()
  print(text)

Command Line:
  python run_web.py
  # Or
  python app_advanced.py

Docker:
  docker-compose up
  # Visit http://localhost

🔧 CONFIGURATION
═══════════════════════════════════════════════════════════════════════════

Environment Variables (.env file):
  FLASK_DEBUG=True          (Enable debug mode)
  PORT=5000                 (Server port)
  MAX_PDF_SIZE_MB=100       (File size limit)
  LOG_LEVEL=INFO            (Logging level)

Flask Configuration (config.py):
  UPLOAD_FOLDER             (Upload directory)
  OUTPUT_FOLDER             (Output directory)
  MAX_CONTENT_LENGTH        (Request size limit)
  ALLOWED_EXTENSIONS        (File types)

🆘 SUPPORT
═══════════════════════════════════════════════════════════════════════════

Troubleshooting:
  • Check logs/app.log for errors
  • Review error messages in UI
  • Validate PDF files
  • Check system resources

Common Issues:
  • Port already in use → Use different port
  • File too large → Check size limit
  • PDF corrupted → Validate file
  • Memory issues → Process fewer files

Getting Help:
  1. Read relevant .md documentation
  2. Check error logs
  3. Review examples/
  4. Check docstrings in source code

📞 QUICK REFERENCE

File Requirements:
  Format:     PDF only (.pdf)
  Size:       Max 100 MB
  Multiple:   Supported (unlimited)

Processing Options:
  Extract Text       Extract document text
  Extract Metadata   Get file information
  Extract Tables     Extract structured data
  Extract Images     Export embedded images

API Endpoints:
  GET /                       Upload page
  POST /api/upload            Upload files
  POST /api/process           Process files
  POST /api/merge             Merge PDFs
  POST /api/split             Split PDF
  POST /api/rotate            Rotate pages (advanced)
  POST /api/info              Get PDF info (advanced)
  GET /api/health             Health check (advanced)
